# データの型

## ルータ

### ルータの Input

> アイデアなので日本語。

```ts
type RouterInput = {
    送り元;
    宛先;

    処理用データ;

    宛先用のデータ空間;
    送り元用のデータ空間;
    認識用のデータ空間;
};
```

#### 送り元

```ts
type 送り元 = string;
```

#### 宛先

```ts
type 宛先 = string;
```

#### 処理用データ

```ts
type 処理用データ = {
    type: string;
    data: unknown;
};
```

#### 宛先用のデータ空間

```ts
type 宛先用のデータ空間 = unknown; //未定
```

#### 送り元用のデータ空間

```ts
type 送り元用のデータ空間 = unknown; //未定
```

#### 認識用のデータ空間

```ts
type 認識用のデータ空間 = unknown; //未定
```

### ルータの プラグインの型

```ts
type RouterPlugin = {
    プラグインの認識名;
    ルータ用のデータ空間;
    起動関数;
};
```

#### プラグインの認識名

```ts
type プラグインの認識名 = string;
```

#### ルータ用のデータ空間

```ts
type ルータ用のデータ空間 = unknown; //未定
```

#### 起動関数

```ts
type 起動関数 = {
    start: () => Promise<void> | void;
};
```
